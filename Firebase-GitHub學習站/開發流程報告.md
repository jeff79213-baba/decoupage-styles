### 第四階段：Bug 修復

| 步驟 | 問題 | 修復方式 |
|------|------|----------|
| 1 | JS 字串中有 3 處未跳脫的 ASCII 單引號（`'main'`、`'feature/add-login'`、`'my-project'`），導致整個 `<script>` 解析失敗，所有功能失效 | 改用中文引號 `「」` |
| 2 | mod1 的 5 題問答各自使用不同的 quizGroup（1~5），導致記分系統無法運算 | 統一使用 `group=1` |
| 3 | mod1 Q1、Q2 的 `data-answer` 值指向錯誤答案 | 修正為正確的索引值 |

## 🛠️ 技術細節

- **語言**：純 HTML5 + CSS3 + Vanilla JavaScript（無任何框架或套件）
- **字體**：透過 Google Fonts CDN 引入 Inter 字型
- **互動機制**：
  - Tab 切換：JS 監聽 click 事件，切換 `.module` 的 `display` 狀態
  - 模擬操作：逐行 `innerHTML` 插入 + CSS `transition` 動畫
  - 問答系統：`data-answer` 屬性記錄正確答案索引，點擊後比對並顯示解析
  - 樹狀圖展開：`toggleTree()` 切換 `.collapsed` class
  - 學習進度條：追蹤已訪問的模組數量，計算百分比
