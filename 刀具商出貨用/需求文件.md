# 刀具商出貨管理系統 - 需求文件

## 專案概述
為刀具商建立出貨管理系統，主要處理：
1. 刀具銷售（自製刀具）
2. 磨刀服務（客戶送刀來磨）

## 業務流程

### 賣刀流程
```
進貨/備貨 → 接訂單 → 出貨（扣庫存）
```

### 磨刀流程
```
客戶送刀 → 磨完 → 出貨（不影響庫存）
```

## 功能需求

### 1. 客戶資料管理
- 客戶名稱
- 電話
- 地址（出貨地址）
- 備註

### 2. 刀具資料管理（SKU）
- 刀具名稱/型號
- 規格欄位（待提供表格）
- 庫存數量
- 單位（支/把）
- 單價
  - 自製刀：首次手動輸入，之後同款套用
  - 磨刀：基本固定價 + 刀刃另計（待討論）

### 3. 出貨單管理
#### 主表欄位
- 出貨單號（自動編號）
- 客戶（選擇，自動帶入地址）
- 出貨日期
- 類型（賣刀/磨刀）
- 備註
- 送貨地址
- 金額顯示設定（客戶可選擇顯示/隱藏）

#### 明細欄位（可動態新增多筆）
- 刀具（從資料庫選擇）
- 數量
- 備註

### 4. 庫存管理
- 賣刀：做完刀→增加庫存，出貨→扣庫存
- 磨刀：不影響庫存

### 5. 出貨單列印
- 需要列印功能
- 之後需連接印表機

### 6. 搜尋/篩選
- 刀具搜尋功能
- 下拉選單選擇刀具
- 具體設定待刀具內容確認後再設計
