# 自訂計算機網頁 — 企劃書

## 專案概述

將現有的「饕之鄉小籠湯包計算機」改造成一個**通用型自訂計算機網頁**。使用者（部署者）只需打開一個網址，即可在網頁上自行設定品項名稱、單價、顏色等樣式，所有設定僅儲存在自己的瀏覽器中，不影響他人。

---

## 目標使用者與角色關係

```
┌─────────────────────────────────────────────────────────┐
│                        你（部署者）                       │
│                   GitHub Pages 放置頁面                   │
│                    提供一個網址給所有人                    │
└────────────────────────┬────────────────────────────────┘
                         │
                         ▼
              ┌──────────────────────┐
              │   單一靜態網頁 HTML   │
              │   （無伺服器、無後端） │
              └──────────────────────┘
                         │
            ┌────────────┼────────────┐
            ▼            ▼            ▼
        朋友 A        朋友 B        朋友 C
      (各自設定)     (各自設定)     (各自設定)
      localStorage  localStorage  localStorage
```

### 角色關係說明

| 角色 | 身份 | 權限與行為 |
|------|------|-----------|
| **你（部署者）** | 頁面的發布者 | 將 HTML 上傳到 GitHub Pages，產生一個公開網址 |
| **終端使用者** | 拿到網址的任何人 | 打開網址 → 在**自己的瀏覽器**中自訂品項、名稱、顏色 → 使用計算機 |
| **軟體（本頁面）** | 純前端工具 | 只提供功能與介面，**不儲存任何資料到伺服器**，所有資料僅在使用者自己的 localStorage 中 |
| **GitHub Pages** | 靜態託管服務 | 只負責存放 HTML 檔案，不參與任何邏輯或資料儲存 |

### 關鍵特性

- **資料與程式分離**：程式碼（HTML/CSS/JS）放在 GitHub Pages，每個使用者的設定資料存在各自的瀏覽器
- **互不干擾**：使用者 A 改了品項名稱，使用者 B 完全看不到，因為資料只存在 A 的 localStorage
- **無帳號系統**：不需要註冊、不需要登入，打開就用
- **備份自主**：使用者可下載設定檔（JSON），換電腦或清瀏覽器後再上傳還原

## 功能需求

### 主畫面（計算機模式）

| 功能 | 說明 |
|------|------|
| 動態品項列表 | 根據設定檔渲染品項卡片，每個品項有名稱、單價、單位 |
| 數量加減 | 每個品項有 + / - 按鈕，數量最低為 0 |
| 總金額計算 | 即時計算所有品項數量 × 單價加總 |
| 訂單明細 | 顯示非零數量品項的清單 |
| 全部歸零 | 一鍵重置所有數量 |
| 進入設定 | 點擊 ⚙️ 圖示切換到設定頁 |

### 設定頁面

| 功能 | 說明 |
|------|------|
| 免責提示框 | 明確告知「所有設定僅儲存於本機瀏覽器」 |
| 頁面標題設定 | 輸入框修改主畫面的標題文字 |
| 頁面副標設定 | 輸入框修改副標題文字 |
| 頁尾文字設定 | 輸入框修改頁尾文字 |
| 幣值符號設定 | 例如 $ / ¥ / NT$ |
| 品項管理 | 新增、刪除品項 |
| 品項編輯 | 每個品項可編輯名稱、單價、單位 |
| 顏色設定 | 頁首底色、強調色、按鈕顏色（顏色選擇器） |
| 儲存設定 | 寫入 localStorage，切回主畫面 |
| 恢復預設 | 清空 localStorage，回到內建預設值 |
| 下載設定檔 | 將目前設定匯出為 .json 檔案 |
| 上傳設定檔 | 讀取 .json 檔案，還原設定 |

### 非功能需求

| 項目 | 說明 |
|------|------|
| 部署方式 | GitHub Pages，單一 HTML 檔案，零依賴 |
| 瀏覽器支援 | 現代瀏覽器（Chrome / Safari / Firefox / Edge） |
| 離線可用 | 第一次載入後即使離線也可使用（因為已載入為單頁） |
| 資料安全 | 不蒐集任何使用者資料，不上傳任何內容至伺服器 |
| 響應式設計 | 手機與電腦皆可正常使用 |

## 技術架構

### 檔案結構（單一 HTML）

```
index.html           ← 唯一檔案，包含全部 HTML / CSS / JS
```

### 資料流

```
┌─────────────┐   載入頁面    ┌──────────────────┐
│  localStorage │ ←───────── │  設定物件 (CONFIG)  │
│  (使用者端)   │ ─────────→ │  控制所有渲染      │
└─────────────┘   儲存設定    └──────────────────┘
                                       │
                              ┌────────┴────────┐
                              ▼                 ▼
                        ┌──────────┐     ┌──────────┐
                        │ 主畫面    │     │ 設定頁面  │
                        │ 計算機    │     │ 編輯表單  │
                        └──────────┘     └──────────┘
```

### 設定物件結構

```javascript
{
  title: "饕之鄉小籠湯包",
  subtitle: "點餐計算機",
  footer: "饕之鄉小籠湯包 · 點一點就幫你算好",
  totalLabel: "總金額",
  resetText: "全部歸零",
  emptyText: "還沒有點餐喔",
  currency: "$",
  colors: {
    headerBg: "#e85d2c",
    accent: "#c73e1d",
    button: "#e85d2c"
  },
  items: [
    { id: "item1", name: "湯包", price: 75, unit: "籠" },
    { id: "item2", name: "紅茶", price: 15, unit: "杯" },
    { id: "item3", name: "奶茶", price: 20, unit: "杯" }
  ]
}
```
