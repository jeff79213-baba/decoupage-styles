## 六、命運轉盤遊戲機制

### 6.1 遊戲目的

送 App 之後，用輕鬆遊戲的方式互動，增加影片趣味性，同時讓老闆有機會「回饋」但不強迫。

### 6.2 轉盤選項（共 6 格）

| 選項 | 效果 | 備註 |
|------|------|------|
| 🎉 **免費招待一份餐點** | 老闆請我們吃東西 | 最有效果的選項 |
| 👍 **幫我們按讚 + 訂閱** | 老闆拿出他的手機訂閱頻道 | 你說要加進去的那個 |
| 📢 **幫你打廣告** | 我們在 IG/FB 幫店家宣傳一次 | 老闆賺到 |
| 🍀 **下次帶朋友來光顧** | 承諾再訪，帶更多人 | 真誠互動 |
| 😅 **謝謝參與** | 什麼事都不用做 | 低壓力選項 |
| ⭐ **累積獎金** | 累積到下一集，讓獎項變大 | 增加連貫性 |

> **設計關鍵**：  
> 選項中有「賺到」的（打廣告、帶朋友來），也有「付出」的（招待餐點、按讚訂閱）。  
> 老闆抽到什麼都是「遊戲結果」，不是你開口要求的，心理壓力小很多。

### 6.3 遊戲話術範例

```
小朋友：「老闆～你已經得到我們的免費點餐系統了！
         你要不要順便玩一個轉盤遊戲？有機會中大獎喔！
         如果你不想玩也沒關係～」
老闆：「好啊來啊！」
（轉盤轉起來～）
小朋友：「哇你抽到了ＯＯＯ！！！」
```

### 6.4 轉盤製作方式（用手機即可）

- 選項 1：用 PowerPoint 或 Canva 做一個圓形圖，截圖後放手機裡，現場用「轉盤 App」轉
- 選項 2：買一個實體轉盤（Y拍/蝦皮約 200–500 元），貼上自製選項貼紙
- 選項 3：用紙做一個簡易指針轉盤，更有手工感

## 七、拍攝流程檢查表（限手機拍攝）

### 7.1 拍攝前準備

- [ ] 手機充滿電（最好帶行動電源）
- [ ] 確認手機儲存空間足夠
- [ ] 領夾麥克風（小朋友聲音小，一定要外接麥克風）
- [ ] 輕便腳架（拍穩定畫面，也可手持）
- [ ] 轉盤道具
- [ ] App 已預先準備好樣板（現場只需要輸入菜單）

### 7.2 現場拍攝重點

- [x] **橫拍**（16:9，YouTube 格式）
- [ ] 進店前先拍店門口 + 招牌
- [ ] 拍老闆用計算機的畫面
- [ ] 拍小朋友跟老闆對話（雙方都要入鏡）
- [ ] 拍老闆試用 App 的特寫（臉部表情 + 手機畫面分開錄）
- [ ] 拍轉盤遊戲過程
- [ ] 拍合照
- [ ] 拍空景（店內環境、菜單、客人）— 剪接時很好用

### 7.3 收音提醒

> **手機內建麥克風會錄到很多環境噪音**  
> 建議購買：領夾式麥克風（NT$200–500，插手機耳機孔或 Type-C）

## 八、各平台上架策略

### 8.1 YouTube 主頻道（5–10 分鐘）

完整版影片，包含全部故事流程。

### 8.2 YouTube Shorts（30–60 秒）

只剪「老闆試用 App」+「轉盤遊戲結果」的精華片段，直式 9:16。

### 8.3 Instagram（15–30 秒）

- 直式短影音
- 開頭直接丟 Hook：「我們送了一個免費點餐系統給老闆，結果…」
- IG 互動性強，適合問問題：「你會想試試看嗎？」

### 8.4 TikTok（15–60 秒）

- 節奏更快、字卡更多
- 可以發不同版本（老闆驚訝版、童言童語版）

### 8.5 Threads

- 文字為主 + 照片 1–2 張
- 分享幕後故事、老闆的反應心得
- 適合引發討論：「你覺得小店需要數位點餐嗎？」

## 九、店家選擇指南

### 9.1 合適的店家類型（優先順序）

| 類型 | 原因 |
|------|------|
| ⭐ **小吃攤/路邊攤** | 幾乎沒有點餐系統，反應最真實 |
| ⭐ **早餐店** | 尖峰時間用手寫單，最有感 |
| ⭐ **飲料店** | 選項多、客製化多，點餐容易亂 |
| ⭐ **便當店** | 常有電話訂餐，需要計算總金額 |
| ✅ **麵店/滷味攤** | 點選項多，計算機常用 |
| ❌ **連鎖品牌** | 已有系統，不適用 |

### 9.2 進店話術

```
小朋友：「老闆您好～我們是一個 YouTube 頻道，
         我叫 OOO，今年 O 歲！
         我剛剛看到你在用計算機算錢，
         我們可以幫你做一個免費的點餐 App 喔！」
```

### 9.3 如果被拒絕

- 老闆可能忙、或不想上鏡頭
- 拍下拒絕過程，剪進影片開頭當鋪陳
- 保持禮貌，說「沒關係，謝謝老闆！」就離開
